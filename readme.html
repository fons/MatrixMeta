Introduction

MatrixMeta is a Scala API around a set of matrix libraries. It uses Scala implicits to enable the user to switch between different implementations.

The goal is to provide a uniform api for both dense and sparse matrices as well as different packages

Building

 
sbt compile

Testing

 
sbt test
